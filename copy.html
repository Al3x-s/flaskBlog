Binary To Decimal

Example:

To convert an 8-digit binary number into a decimal number, you can use the positional value system.
Each digit in the binary number represents a power of 2 based on its position.
Let's take the 8-digit binary number 11011010 as an example:
1 * 2^7 (1 multiplied by 2 raised to the power of 7, which is 128)
+ 1 * 2^6 (1 multiplied by 2 raised to the power of 6, which is 64)
+ 0 * 2^5 (0 multiplied by 2 raised to the power of 5, which is 0)
+ 1 * 2^4 (1 multiplied by 2 raised to the power of 4, which is 16)
+ 1 * 2^3 (1 multiplied by 2 raised to the power of 3, which is 8)
+ 0 * 2^2 (0 multiplied by 2 raised to the power of 2, which is 0)
+ 1 * 2^1 (1 multiplied by 2 raised to the power of 1, which is 2)
+ 0 * 2^0 (0 multiplied by 2 raised to the power of 0, which is 0)
Adding these values together, we get: 128 + 64 + 0 + 16 + 8 + 0 + 2 + 0 = 218
So, the 8-digit binary number 11011010 is equal to 218 in decimal form.
